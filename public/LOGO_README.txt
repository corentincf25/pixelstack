LOGO PIXELSTACK
===============

Pour que ton logo s'affiche dans la sidebar :

1. Copie ton fichier image (PNG) dans CE dossier (public/).
2. Renomme-le EXACTEMENT : logo.png
   - Tout en minuscules : logo
   - Extension : .png
   - Sur Vercel/Linux le chemin est sensible a la casse (Logo.PNG ne marchera pas).

3. Chemin final : mini-maker-app/public/logo.png

4. Redémarre le serveur dev (npm run dev) puis recharge la page (Ctrl+F5).

Si le fichier est absent ou mal nomme, la sidebar affiche un bloc violet/bleu a la place.
